# UI 90 Upgrade Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Upgrade the shared shell and seven core Nerdy pages to a consistent 90+ quality bar while preserving real session, model, and avatar functionality.

**Architecture:** Start with the shared shell and reusable page primitives, then refit each page onto those primitives. Keep functional logic intact where it is already real, and concentrate changes on information architecture, presentation, honest controls, and missing learner-facing features.

**Tech Stack:** Next.js app router, React, TypeScript, global CSS, Testing Library, pnpm
